Windows 11

Windows 11 se lanzó de forma oficial el 5 del 2021. Ha llegado a modo de actualización gratuita para los PCs con Windows 10 original que lo soporten
Si la actualización gratuita no te llega, tienes varias opciones para forzar su instalación. Para eso, tienes que ir a la web Microsoft.com/es-es/software-download/windows11 donde vas a tener tres opciones o maneras diferentes en las que descargar Windows 11:
1-Asistente de instalación de Windows 11: Se trata de un archivo ejecutable que puedes utilizar en cualquier ordenador con Windows, y lanza un asistente que te guía de forma sencilla por todo el proceso de instalación
2-Crear soportes de instalación de Windows 11: Descargas un archivo llamado MediaCreationToolW11.exe. Con él, vas a poder crear un USB o DVD de instalación de forma sencilla y paso a paso. Luego, este USB lo podrás conectar a cualquier ordenador para instalar Windows 11 en él.
3-Descargar imagen de disco de Windows 11 (ISO): Descargarás una imagen ISO de Windows 11. Luego, podrás montar la imagen ISO en cualquier USB de forma manual, y usarlo para iniciar e instalar Windows 11 en cualquier ordenador que quieras.
De estos tres, los dos primeros métodos son los más sencillos, ya que funcionan mediante programas que hacen de asistentes, guiándote paso a paso por el proceso para instalar Windows 11 o para crear una unidad con la que puedas instalarlo. El tercero es un buen método si estás utilizando otro sistema operativo, pero quieres tener la ISO para preparar una unidad con la que instalar Windows 11 a un lado o en otro ordenador.

El camino hasta Windows 11

Windows desde su lanzamiento ha recibido múltiples cambios de apariencia y funcionalidades durante los años. En esta galería verás una breve muestra de la evolución de la interfaz gráfica desde su versión 1.0.

Windows 1.0 (1985)

Interfaz gráfica basada en ventanas. Corría sobre el sistema operativo MS-DOS.Trató de mejorar la interacción del usuario e introducir un concepto primigenio de multitarea.

Windows 2.0 (1987)

Mejoraron los gráficos, se amplió la memoria, se introdujeron los iconos de escritorio, un panel de control, botones para maximizar y minimizar las ventanas y la posibilidad de superponerlas. Aparecen las primeras versiones de Word y Excel.

Windows 3 (1990)

Microsoft introdujo mejoras en el rendimiento y en la multitarea. También se renovaron los iconos del escritorio. Se introduce el Explorador de Archivos y el Administrador de Programas.

Windows NT (1993)

Fue un sistema operativo de 32 bytes multitarea y multiusuario. Fue el primer S.O en usar Unicode internamente. Diseñado para correr en múltiples arquitecturas, con el núcleo separado del hardware.

Windows 95 (1995)

Renovada interfaz gráfica y de usuario. También se incluyeron el navegador Internet Explorer, el botón de inicio, la barra de tareas y el área de notificaciones.

Windows 98 (1998)

Versión mejorada de Windows 95, se diferenciaba de ésta sobre todo por un nuevo sistema de archivos más rápido y con más capacidad. También da comienzo al gran auge de Internet y era compatible con los DVD y dispositivos USB.

Windows Millennium Edition (2000)

Recibió duras críticas debido a sus problemas de estabilidad y seguridad y supuso un fracaso para Microsoft. Estaba orientado al mercado profesional.

Windows XP (2001)

Una de las versiones más exitosas de Windows, teniendo soporte hasta el 2014. Espectacularrediseño, con una estética limpia, y un funcionamiento eficaz y con pocos errores. Aparece laposibilidad de utilizar varias cuentas de usuario a la vez.

Windows Vista (2007)

Recibió una nueva interfaz gráfica llamada Aero, que permitía nuevos efectos y la transparencia de las ventanas, pero esto provocó problemas de rendimiento en ordenadores menos potentes.

Windows 7 (2009)

Versión más sólida de windows. Se implementaron mejoras en el rendimiento, estabilidad y experiencia de usuario, se introduce una nueva barra de tareas y Windows Touch, haciéndolo compatible con pantallas táctiles.

Windows 8 (2012)

Eliminación del botón de inicio, la finalidad era unificar la experiencia de usuario para distintos dispositivos: ordenadores, Tablets y Smartphones. Introdujo la nueva interfaz de usuario Metro UI. No tuvo éxito.

Windows 10 (2015)

Se caracterizó por lanzar actualizaciones masivas, añadiendo mejoras y nuevas funciones, la vuelta del menú de inicio, el nuevo navegador Microsoft Edge, el asistente de voz Cortana y la sincronización con la nube.

Discos HDD, SSD y M.2

Discos HDD

El hecho de que tenga que desplazarse el lector mecánicamente hacia una posición y otra hasta llegar al lugar adecuado, hace que resulten más lentas las cargas y aperturas de los distintos programas, de ahí que hubiese una necesidad de crear formatos más rápidos y compactos.

Discos SSD

Se ha eliminado el disco de lectura y el brazo mecánico. Pasamos a la memoria flash que es la misma que se utiliza en los pendrive y en las tarjetas SD. Este tipo de memoria ya no necesita un lector que vaya a buscar la información, por lo que los accesos y lecturas son mucho más rápidos.

Discos M.2

Son discos duros sólidos con gran aumento en la velocidad. Se conectan a través de los puertos PCIe, lo que les da mucha más velocidad y amplitud de transferencia. Los puertos PCIe son los mismos que utilizan componentes como la tarjeta gráfica o la tarjeta de red, entre otras.
